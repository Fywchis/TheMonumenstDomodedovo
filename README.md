КАК УСТАНОВИТЬ

1. СКОПИРОВАТЬ РЕПОЗИТОРИЙ:
   git clone <your-repo-url>
   cd my_project
   
2. УСТАНОВИТЬ ЗАВИСМОСТИ:
```
pip install -r requirements.txt
```
or
```
pip install .
```
3. ЗАПУСТИТЬ КОД:
```
python main.py
```
(ОПЦИОНАЛЬНО) ИСПОЛЬЗУЯ CLI:
```

```
